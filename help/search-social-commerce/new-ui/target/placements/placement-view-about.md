# (Nieuwe interface) Informatie over de weergave [!UICONTROL Placements]

*eigenschap van Beta*

De weergave [!UICONTROL Manage] > [!UICONTROL Placements] bevat twee tabbladen:

* Het tabblad [!UICONTROL Placements] bevat alle plaatsen in de gefilterde weergave voor het geselecteerde adverteerderaccount.

* Het tabblad [!UICONTROL Negatives] bevat alle negatieve plaatsingen in de gefilterde weergave voor het geselecteerde adverteerderaccount.
