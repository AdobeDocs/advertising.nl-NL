# (Nieuwe interface) Informatie over de weergave [!UICONTROL Keywords]

*eigenschap van Beta*

De weergave [!UICONTROL Manage] > [!UICONTROL Keywords] bevat twee tabbladen:

* Het tabblad [!UICONTROL Keywords] bevat alle trefwoorden in de gefilterde weergave voor het geselecteerde adverteerderaccount.

* Het tabblad [!UICONTROL Negatives] bevat alle negatieve trefwoorden in de gefilterde weergave voor het geselecteerde adverteerderaccount.
